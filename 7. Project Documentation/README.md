# Project Documentation

## Project Overview

Voice-Based Concept Understanding Analyzer (VBCUA) is an AI-powered educational application that evaluates a student's conceptual understanding from spoken explanations. The system converts speech into text using OpenAI Whisper, compares it with a reference answer using Sentence Transformers, calculates a semantic similarity score, generates grades and feedback, and produces a downloadable PDF report.

---

## Objectives

- Automate concept evaluation using Artificial Intelligence.
- Reduce manual effort in student assessment.
- Provide accurate semantic analysis of spoken responses.
- Generate instant feedback and assessment reports.
- Improve learning outcomes through automated evaluation.

---

| PDF Generation | ReportLab |

## Modules

1. Speech-to-Text Conversion
2. Semantic Similarity Analysis
3. Concept Scoring
4. Feedback Generation
5. PDF Report Generation
6. Streamlit User Interface

---

## Project Workflow

1. Upload audio file.
2. Convert speech into text.
3. Load reference answer.
4. Perform semantic similarity analysis.
5. Calculate concept understanding score.
6. Generate grade and personalized feedback.
7. Generate downloadable PDF report.

---

## Outcome

The project successfully automates concept understanding assessment using AI technologies. It provides quick, accurate, and reliable evaluation of spoken answers while generating professional assessment reports.
