# Project Demonstration

## Project Title

**Voice-Based Concept Understanding Analyzer (VBCUA)**

---

## Project Demonstration

This project demonstrates an AI-powered application that evaluates a student's conceptual understanding from spoken responses.

The application performs the following tasks:

- Upload an audio file (WAV, MP3, or M4A)
- Convert speech to text using OpenAI Whisper
- Compare the transcript with a reference answer using Sentence Transformers
- Calculate the semantic similarity score
- Generate a concept understanding score
- Provide grade and personalized feedback
- Generate a downloadable PDF assessment report

---

## Demonstration Steps

1. Launch the Streamlit application.
2. Upload a supported audio file.
3. Load the reference answer.
4. Click **Analyze**.
5. View the transcript.
6. Check the similarity score and generated feedback.
7. Download the PDF report.
